# Назначение файлов

- `main.py` – точка входа, создает FastAPI-приложение и подключает роуты.
- `router.py` – описывает HTTP эндпоинты и связывает их с логикой.
- `models.py` – ORM-модели SQLAlchemy для работы с БД.
- `schemas.py` – Pydantic-схемы для валидации входных/выходных данных.
- `database.py` – подключение к БД, сессии и зависимости для FastAPI.
- `config.py` – загрузка переменных окружения и других настроек.
- `constants.py` – статические значения, которые не меняются (типы, пути и т.п.).
- `exceptions.py` – кастомные исключения и обработка ошибок.
- `utils.py` – вспомогательные функции, не относящиеся к бизнес-логике.
- `service.py` – бизнес-логика, отдельно от API и моделей.
